# apiopenmeteo-kafka-spark-postgresql-streamlit

🌍 Real-Time Weather Intelligence
☀️ Sunshine • 🌧️ Rainfall • ❄️ Snow Events Analytic

🏗️ Architecture
Open-Meteo API → Kafka → Spark → PostgreSQL → Streamlit / Jupyter

📡 Data Source: Open-Meteo API

📨 Streaming: Kafka

⚙️ Processing: Spark (PySpark)

🗄️ Storage: PostgreSQL

📊 Visualization: Streamlit & Jupyter
